# Web Design 

## Description
This website is designed to be an brief and basic introduction to salamanders while showing off the skills I have learned so far in Intro to Web Design. 
 
 
## Features
**note: each feature may have been used more than once but only one is documented here. 

Header:containting the set html meta data charset, name, content and title. I have also included a link to the "stylepage.css" that dictates the rest of the pages design.

Footer: On the home page my UVM email has been added.

Article: This containts the paragraph on page 2 describing the species overview. 

Section: On page 2, the flexboxes the containing table head and table data of genus and species and the links to navigate to the other pages.  

Nav: On page 2, within the section described above, nav contains the flexbox content that allows for going to different pages of the webside. 

Headings: Each page has a heading at the top of the page describing the content within that page. 

Paragraphs: On page 2 a paragraph contains the species overview as well as a link. 

Lists: On page 3 there is a list containing the environmental imporatnce salamanders have. 

Links: There is a link on pages 3, labeled click me, that will send you to the New England Zoo website where you can learn more about conservation efforts.

Images: On the homepage there are 2 pictures of the Marbled Salamnder as well as 3 other pictures depicting similar species. 

Table: On page 3 a table contains the table head: genus and species as well as the table data. 

Color Scheme/ background styling: One .css color scheme was set on a seperate page and tagged on every html page to create a consistend yellow background and pink titles.

Typography: I chose didot serif as my font choice, which I kept consistend to make reading it easier. 

Style (lists, tables, text properties): For the lists talking about the threats salamanders face, the title has a grey background and is horizontal instead of verticle. Each list title is bold and underlined. My table head head, for the genus and species, has a blue background. The table data is italicised. 

Margin/padding/ borders: The paragagraph on "Species Overview" page has a padding of of 20 px. On page 3- The list containing salamander threats has a margin-left and right of 150 px. The flexbox containing both the picture, threats list, and conservation effort description has a black border of 1 px. 

Flexbox: On the homepage, a flexbox container holds the links to navigate the website and two marbled salamander pictures. 

Media query: In the style.css, there is a media query for flexbox-container1.
